# Rainydays

![image](https://user-images.githubusercontent.com/52622303/164316813-4b12d99f-aeb7-4069-85cf-e72b3a50ac99.png)

Online store for rain jackets! The product of a cross course project from my first semester of Front-End Development studies at Noroff Vocational School.

## Description

The design is self made from an earlier course, and the coding languages for this course was limited to HTML and CSS.
The purpose of this project is to practice the following:

- Understanding and practical knowledge in coding with HTML and CSS
- Responsive, adaptive, and fluid design
- Universal design
- Prototyping with HTML
- Semantically correct HTML
- CSS Media Queries
- User-oriented testing

## Technologies

- HTML
- CSS
- Prettier

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone https://github.com/tominehaug/Rainydays.git
```
Run this command in the Visual Studio Code Terminal.

2. Install the dependencies:

No dependencies for this project.

### Running

To run the app use LiveServer extension in Visual Studio Code.
